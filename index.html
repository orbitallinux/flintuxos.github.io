What is Orbital Linux?

Orbital Linux is a linux distribution following the stable release (LTS) cycle of the monolithic linux kernel. It is made primarly for systems that don't need to be updated every now and then, which could be benefitial for certain applications.

Project's state

However, its development is only being done by one person (which is me) and it's still really early in development to be a final product at the moment.
